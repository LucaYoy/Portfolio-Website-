# **App Name**: LucaPortofolio

## Core Features:

- Profile Display: Display name, current position, and a professional profile photo.
- Bio Section: Showcase a narrative bio with academic and industry framing.
- Publications Display: Display publications with titles, authors, venues, and links to arXiv, journal, or PDF.
- Projects Showcase: Display academic and personal projects with descriptions and links to GitHub or demos.
- Contact Information: Provide external links to email, LinkedIn, Google Scholar, and GitHub.
- Layout Adaptation: Adjust layout dynamically for different screen sizes, with a focus on mobile devices.

## Style Guidelines:

- Dark background: Dark navy (#1A232E) for a tech-forward and professional feel.
- Primary color: Electric blue (#7DF9FF) to add visual interest and modern style. Avoids a tech cliche while still resonating with the industry.
- Accent color: Vivid violet (#D070E0) for subtle highlights and active states.
- Body font: 'Inter', sans-serif, for a modern, machined, neutral look and improved readability.
- Headline font: 'Space Grotesk', sans-serif, for a computerized, techy, scientific feel.
- Simple, outline-style icons for external links and navigation.
- Single-page layout with smooth scrolling and a fixed top navigation bar.
- Subtle fade-in animations on content sections as the user scrolls.
